# ansible

## Xcodeのインストール
$ sudo xcodebuild -license

## Homebrewのインストール
$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

## Ansibleのインストール
$ brew doctor

$ brew update

$ brew upgrade

$ brew install python

$ brew install ansible

## Ansibleの実行
$ ansible-playbook mac.yml -i hosts

## alfred link
$ brew cask alfred link
